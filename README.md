# A股自选股资金异动监控

只盯自选池,不扫全市场。基于 AKShare + Streamlit,零成本运行。

## 功能

- 📌 **自选池管理**:添加/删除/导出股票,本地 JSON 持久化
- 📊 **实时资金流**:主力 / 超大单 / 大单 / 净流入占比
- ⚡ **异动识别**:4 类规则自动打标签(强流入 / 抢筹 / 出货 / 拉升出货)
- 📈 **历史趋势**:个股近 30 日资金流堆叠图 + 数据明细
- 🔄 **缓存策略**:60 秒缓存,避免高频拉接口被限流

## 一键运行

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 启动
streamlit run app.py

# 3. 浏览器会自动打开 http://localhost:8501
```

## 自选池数据

存在当前目录的 `watchlist.json`,首次运行会创建 5 只默认股票(茅台、宁王、比亚迪、平安、五粮液),你可以自由替换。

格式:
```json
[
  {"code": "600519", "name": "贵州茅台"},
  {"code": "300750", "name": "宁德时代"}
]
```

## 异动规则

| 标签 | 触发条件 | 含义 |
|------|---------|------|
| **强流入** | 主力净流入 ≥ 1 亿 且 占成交额 > 5% | 大资金扫货 |
| **抢筹** | 超大单 ≥ 5000 万 且 占主力净额 > 60% | 机构主导 |
| **出货** | 主力净流出 ≥ 5000 万 且 跌幅 > 2% | 大资金离场 |
| **拉升出货?** | 涨幅 > 3% 但主力净流出 | 警惕诱多 |

阈值在 `app.py` 的 `classify_anomaly()` 函数里,你可以按自己的风格改。

## 数据口径说明

- 数据源:东方财富网(通过 AKShare 封装)
- "主力" = 超大单(单笔≥100万)+ 大单(20-100万)
- **基于单笔成交金额估算,不是真实机构资金**
- 免费数据是 3 秒切片(非逐笔),与同花顺/Choice 口径会有差异
- 非交易时段显示的是收盘后最后一次更新数据
